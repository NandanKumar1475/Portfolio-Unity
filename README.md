# Nandan Kumar XR Portfolio

Production-focused recruiter portfolio for Unity XR Developer and AR/VR Simulation Engineer roles.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui-style local components
- lucide-react

## Commands

```bash
npm run dev
npm run typecheck
npm run build
```

## Deploy

Deploy on Vercel with the default Next.js settings.

Before going live, update placeholder social links and email in `src/data/socials.ts`.
